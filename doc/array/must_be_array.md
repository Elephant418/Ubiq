[Ubiq](../index.md) / [String](../index.md#array) / must_be_array
======


Description
-------- 

```php
void \Util\Arr\must_be_array( mixed &$array );
```

If the parameter is not an array, it is encapsulated to be an array.

**Note**: The array is passed by reference.
